关于我

你好，我是林一，一名在北京工作的软件工程师，目前主要做 Java 后端开发，业余时间喜欢折腾前端。

写博客是我整理思路的方式。每次深入研究一个技术问题，把它写成一篇清晰的文章，往往比写代码本身花的时间更多——但也让我理解得更深。

除了写代码，我喜欢读书（每年大约二十本）、爬山、摄影，偶尔也会一个人去远处旅行。生活和技术对我来说并不割裂，它们都是理解这个世界的方式。

如果你读到了让你有共鸣的文章，或者有想探讨的话题，欢迎给我发邮件。

技术栈

写作方向

这个博客大致分为几个方向：

技术文章：以 Java 后端和前端 Vue 为主，记录工作中遇到的有意思的技术问题、解决方案，以及一些深入研究的主题。

工具与工程化：开发工具、工程化配置、效率提升，对搭建好用的开发环境有执念。

读书笔记：技术书和非技术书都读，会选择有话说的写成文章。

生活随笔：旅行见闻、年度总结，代码之外的那些事。
